Welcome to XACML Inspector

This tool analyses XACML policies by transforming them into an OWL ontology and a Prolog representation. Via a web interface, the transformed policy can be downloaded and it can be analysed for questions such as

  • is the policy consistent?
  • which attributes are required to answer a request?
  • which requests would lead to a certain decision?

Getting started

To get started with XACML inspector, you will first need SWI-Prolog. Get it here and install it.

Then, checkout the eclipse project in this repository:

$ mkdir ~/xacmlinspect
$ cd ~/xacmlinspect
$ git clone git://github.com/quadriat/XACML-Inspect.git

Open eclipse and import the project PolicyAnalysisTool from ~/xacmlinspect/PolicyAnalysisTool into your workspace.
